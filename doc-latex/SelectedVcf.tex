\section{SelectedVcf} \label{sec:SelectedVcf}

This utility creates a new coordinate file containing only beads of specified
types.

Using \tt{--reverse} switch, the bead types can also be specified in reverse,
i.e., bead types to be excluded from the output file(s).

Which timesteps are saved can be controlled using \tt{-st} option for
the first timestep to save and \tt{-e} option for the last timestep to
save; using \tt{-sk <int>}, \tt{<int>} steps are ignored per one
step saved. If the option to save only specified timesteps (\tt{-n}) is
used, \tt{-sk} is ignored. A maximum of 100 steps can be explicitly
specified using the \tt{-n} option.

There is also an option to remove periodic boundary conditions (i.e., to join
molecules). Conversely, the simualation box can be wrapped (i.e., the
periodic boundary conditions applied). If both \tt{--join} and
\tt{-w} options are used, the simuation box is first wrapped and then
the molecules are joined.

Lastly, the \tt{--last} saves to output file(s) only the last step from
the input coordinate file; it is useful, when only the last timestep is
required, but the total number of timesteps in the input file is unknown.

Usage:

\vspace{1em}
\noindent
\tt{SelectedVcf <input> <output> <bead type(s)> <options>}

\vspace{1em}
\noindent
\begin{longtable}{p{0.22\textwidth}p{0.724\textwidth}}
  \toprule
  \multicolumn{2}{l}{Mandatory arguments}\\
  \midrule
  \tt{<input>}        & input coordinate file\\
  \tt{<output.vcf>}   & output coordinate file with\\
  \tt{<bead type(s)>} & bead type names to save (can be omitted if
    \tt{--reverse} is used)\\
  \toprule
  \multicolumn{2}{l}{Options} \\
  \midrule
  \tt{--reverse} & reverse bead types to save, i.e., exclude \tt{<bead type(s)>}
    instead of including them; if no \tt{<bead type(s)>} are specified, all bead
    types are saved\\
  \tt{--join}    & join molecules by removing periodic boundary conditions\\
  \tt{--wrap}    & wrap simulation box (i.e., apply periodic boundary
    conditions)\\
  \tt{-st <n>}   & starting timestep for calculation (default: 1)\\
  \tt{-e <n>}    & ending timestep for calculation (default: none)\\
  \tt{-sk <n>}   & number of steps skip per one used (default: 0)\\
  \tt{-n <int(s)>} & save only specified timesteps\\
  \tt{--last} & save only the last step\\
  \tt{-x <name(s)>} & exclude molecules of specified name(s) -- do not
    use if \tt{<output.vcf>} is further analysed \\
  \tt{-xyz <name>} & save coordinates to \tt{xyz} file -- does not
    take into account \tt{-x} option \\
  \bottomrule
\end{longtable}
